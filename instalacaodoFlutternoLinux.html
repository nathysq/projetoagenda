Tutorial de Flutter - Parte 1

Aprenda a fazer mobile apps para iOS e Android usando Flutter!

Antes de tudo, o que é Flutter?

Flutter é um framework de rápido desenvolvimento de aplicativos mobile expressivos e com interface gráfica flexível para Android e iOS, compilados em código nativo.

Características:

  • Desenvolvimento rápido

    • Flutter possui uma ótima ferramenta para desenvolvimento rápido: hot reload.
    Com o hot reload você consegue fazer mudanças nas widgets do Flutter em tempo de execução sem precisar recompilar o aplicativo.


  • Interface gráfica expressiva e flexível

    • Envie rapidamente os recursos com foco nas experiências nativas do usuário final. A arquitetura em camadas permite personalização completa, o que resulta em renderização incrivelmente rápida e designs expressivos e flexíveis.

  • Desempenho nativo

    • Os widgets do Flutter incorporam todas as diferenças críticas de plataforma, como rolagem, navegação, ícones e fontes, para fornecer desempenho nativo completo no iOS e no Android.



Instalação

Instalação no Linux

Requisitos de Sistema

Para instalar e rodar o Flutter, seu ambiente de desenvolvimento precisa ter no mínimo:

  • Sistema operacional: Linux (64-bit)
  • Espaço de disco: 600 MB
  • Ferramentas
    • Flutter depende dessas ferramentas de linha de comando disponíveis no seu ambiente de desenvolvimento:
    • bash, mkdir, rm, git, curl, unzip, which

Download Flutter SDK

  1. Faça o download do seguinte pacote de instalação para obter a versão beta mais recente do Flutter SDK (para outros canais de lançamento e versões mais antigas, consulte a página de arquivos do SDK.):
    • flutter_linux_v0.8.2-beta.tar.xz

  2. Extraia o arquivo no local desejado, por exemplo: 
    $ cd ~/development
$ tar xf ~/Downloads/flutter_linux_v0.8.2-beta.tar.xz
  3. Adicione o flutter ao PATH: 
    $ export PATH=`pwd`/flutter/bin:$PATH

O comando acima define temporariamente sua variável PATH para a janela do terminal atual. Para adicionar permanentemente o Flutter ao seu PATH, consulte Atualizar seu PATH mais à frente.

Agora você está pronto para executar os comandos Flutter!


Execute 'flutter doctor'

Execute o seguinte comando para ver se há alguma dependência que você precisa instalar para concluir a instalação:

   $ flutter doctor

Este comando verifica seu ambiente e exibe um relatório para a janela do terminal. O Dart SDK é empacotado com Flutter; não é necessário instalar o Dart separadamente. Verifique cuidadosamente a saída de outro software que você possa precisar instalar ou outras tarefas para executar (mostradas em negrito).

Por exemplo:

    [-] Android toolchain - develop for Android devices
        • Android SDK at /Users/obiwan/Library/Android/sdk
        ✗ Android SDK is missing command line tools; download from https://goo.gl/XxQghQ
        • Try re-installing or updating your Android SDK,
          visit https://flutter.io/setup/#android-setup for detailed instructions.

As seções a seguir descrevem como executar essas tarefas e finalizar o processo de instalação.

Depois de instalar as dependências ausentes, execute o comando flutter doctor novamente para verificar se você configurou tudo corretamente.


Atualizar seu PATH

Você pode atualizar sua variável PATH para a sessão atual somente na linha de comando, conforme mostrado em Download Flutter SDK. Você provavelmente desejará atualizar essa variável permanentemente, para poder executar comandos de flutter em qualquer sessão de terminal.

As etapas para modificar permanentemente essa variável para todas as sessões de terminal são específicas da máquina. Normalmente, você adiciona uma linha a um arquivo que é executado sempre que você abre uma nova janela. Por exemplo:

  1. Determine o diretório em que você colocou o Flutter SDK. Você precisará disso no Passo 3.
  2. Abra (ou crie) $HOME/.bash_profile. O caminho do arquivo e o nome do arquivo podem ser diferente na sua máquina.
    (No Ubuntu 16.04 é o arquivo $HOME/.bashrc.
  3. Adicione a seguinte linha e altere [PATH_TO_FLUTTER_GIT_DIRECTORY] para o caminho onde você clonou o repositório git do Flutter:
   $ export PATH=[PATH_TO_FLUTTER_GIT_DIRECTORY]/flutter/bin:$PATH

  1. Execute o source $HOME/.bash_profile para atualizar a janela atual.

  2. Verifique se o diretório flutter/bin está agora em seu PATH executando:

   $ echo $PATH

Se tudo estiver OK, você está pronto para usar o Flutter!
